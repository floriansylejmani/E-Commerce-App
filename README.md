# E-Commerce App

## Project Overview
This E-Commerce App is designed to provide a seamless online shopping experience. It offers users the ability to browse products, manage their cart, and make purchases securely. The application aims to enhance user engagement and improve conversion rates through a user-friendly interface.

## Technology Stack
- **Frontend:** React.js, Redux, Bootstrap
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Payment Gateway:** Stripe
- **Hosting:** Heroku / AWS

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/floriansylejmani/E-Commerce-App.git
   ```
2. Navigate into the project directory:
   ```bash
   cd E-Commerce-App
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```
